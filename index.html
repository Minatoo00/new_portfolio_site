その経験以来、さまざまなAIツールの情報を集め、実際に試しながら知識とスキルを深めてきました。現在では、AIは私の日常に欠かせない存在となり、開発や学習に積極的に活用しています。

開発スタイル

要件定義・設計書から実装まで、AI支援ツールを活用した仕様駆動開発を実践しています。動くものを作ることは大前提ですが、AIに作ってもらったコードをそのまま使うのではなく、自分でも勉強して理解を深めることを大切にしています。まだ基礎力は十分ではありませんが、コードの品質や保守性・可読性・安全性を意識できるエンジニアになれるよう学習を続けています。

Core Web Vitals への配慮

  • LCP: 最大要素表示時間は2.5秒以下を目標に、画像最適化と初期表示の軽量化を心がけています。
  • CLS: スケルトンUIや固定寸法のコンポーネントを用い、レイアウトのズレを抑えています。
  • INP: ローディング表示や即時フィードバックを徹底し、操作応答の体感を損なわないようにしています。
