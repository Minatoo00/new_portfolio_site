AI駆動開発ツール

日常的に活用しているAI支援ツールと、開発フローへの組み込み方を紹介します。

Cursor

AI IDE / ペアプログラミング

エディタ内でコード生成・リファクタリング・テスト作成までを対話的に完結。リポジトリ全体の文脈を踏まえた変更提案で、スピードと一貫性を両立します。

  • リポジトリ横断の文脈理解と安全な一括編集
  • 型やLintに整合した差分提案と自動修正
  • 設計レビューと代替案の提示（diff比較）

Codex CLI

CLIオートメーション / スキャフォールド

要件を短いプロンプトで記述し、雛形生成・反復実行・差分適用を自動化。プロジェクト横断の共通タスク（生成・変換・検証）をスクリプト化して再現性を高めます。

  • 雛形生成・コード変換・設定ファイル更新の自動化
  • 反復実行しやすいコマンド設計と差分適用
  • CI/CDやローカル開発フローへの容易な組み込み
