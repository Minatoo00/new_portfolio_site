脳神経科学に基づく悩み相談アプリ

脳科学 × AI によるパーソナライズド・アドバイス

ユーザーの自由記述を受け取り、初期アンケート情報と合わせてLLMに解析を依頼し、確認用の簡潔な質問（はい/いいえ）を返すシンプルな仕組みです。数ラウンドのやり取り後、仮説と提案を返します。

プロジェクト期間
2024年中期
役割
フルスタック開発・脳科学研究
プロジェクト種別
個人開発・研究ベース

解決したい課題

従来のメンタルヘルスアプリは表面的なアドバイスが多く、個人の脳機能特性や神経学的状態を考慮していない。以下の課題を解決したいと考えました：

  • 一般的すぎるアドバイスで実効性が低い
  • 科学的根拠が不明確で信頼性に欠ける
  • ユーザー固有の脳機能パターンを無視
  • 短期的対処法のみで根本的改善に至らない
  • 「なぜこのアドバイスなのか」の説明不足

アプローチ

過度に複雑にせず、段階を分けて情報を整理する方針です：

初期アンケートの活用

睡眠/運動/注意などの基礎情報を事前に取得して文脈に反映

確認質問（フォローアップ）

はい/いいえで答えられる具体的な質問を最大6件生成して確認

仮説と簡潔な提案

十分な情報が揃ったら仮説と最大3件の提案を返す

システム構成と主要機能

初回アンケート（Initial Survey）

睡眠・運動・集中傾向などの基礎情報を保存（ユーザー毎に1件）。

  • Next.js App Router + Prisma（SQLite）
  • サーバーAPI: POST /api/survey
  • 最大2件までの期待カテゴリ選択

フィードバック解析（初回）

自由記述を受け取り、アンケート文脈と合わせてLLMに送信。

  • API: POST /api/feedback
  • OpenRouter Chat Completionsを使用
  • JSONのみを抽出し、Zodで最低限バリデーション

フォローアップと仮説生成

はい/いいえ回答を受け取り、必要に応じて次の質問または仮説と提案を返す。

  • API: POST /api/feedback/followup
  • 最大3ラウンドまでの対話履歴を保存
  • 完了時に仮説と最大3つの提案を保存

  • 前頭前皮質：実行機能・意思決定・感情制御

  • 神経新生：新しい神経細胞の生成促進
  • 概日リズム：睡眠・覚醒・ホルモン調節

使用技術

サーバー・データ

Next.js (App Router) Prisma SQLite NextAuth

LLM・フロントエンド

OpenRouter React TypeScript Zod

画面遷移の例

1: サインアップ/ログイン（NextAuth）

2: 初回アンケート（/initial-survey）

3: ダッシュボードで自由記述を送信→初期解析と確認質問

4: 回答送信→必要なら質問継続、十分なら仮説と提案表示

  • 対話は最大3ラウンド（内部制限）
  • 既存履歴の参照/削除が可能

成果と学び

プロジェクトの成果

  • Next.js + Prisma + OpenRouterで最小限のプロトタイプを構築
  • 自由記述→確認質問→仮説/提案 という素直な流れを検証

技術的・学術的な学び

  • LLM出力のJSON抽出と最低限のバリデーション運用
  • はい/いいえ中心の質問設計の扱いやすさ
  • 履歴保存と段階的対話のシンプルな実装

今後の展望・拡張可能性

このアプリは脳神経科学とAI技術を結びつけた実証的研究の第一歩です。今後は生理学的データ（心拍変動、睡眠パターン等）の統合、より精密な個人モデルの構築、臨床心理学・精神医学領域との連携を視野に入れています。「なぜそのアドバイスなのか」を科学的に説明できるAIシステムの可能性を追求し続けたいと考えています。
